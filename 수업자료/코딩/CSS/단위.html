단위

크기 단위

px 단위

- 브라우저(html, root) 기본 설정 : 글자 크기 16px

- 변하지 않는 고정값

- 최소값 1px

% 단위

- 백분율, 가변값

- 부모(!!!!) 요소 크기 기준 특히 width(가로)

- 보통 면적 크기에 사용, 글자 크기에도 사용은 가능

- 사용한 속성(width, font-size)과 같은 부모 속성 값

em 단위

- 배수 단위, 가변값

- 부모 요소의 글자 크기 기준/p>

- 글자 크기에 사용

vw / vh 단위

- viewport(화면) 기준, 가변값

- 화면 크기 == 100px

- 반응형에서 사용, 화면을 기준으로 할 때 사용

rem 단위

- root의 글자 크기를 기준

- 반응형에서 사용, 글자 크기와 면적에도 사용

컬러 단위

관용색명 : 브라우저, 모니터에 따라 달라 보일 수 있다.
HEX 코드값 : #F00
rgb() 함수값 : rgb(255,0,0)
rgba() 함수값 : rgba(0,0,0,0.5)
