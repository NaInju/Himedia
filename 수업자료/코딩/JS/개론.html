웹문서에서 자바스크립트를 사용하는 방법

별도의 js 파일을 연결

  • script 태그에 src 속서에 파일 경로
  • script 태그 내부에는 아무것도 작성하지 말 것

현재 문서에서만 사용

  • script 태그를 사용하여 태그 내부에 JS 문법 작성
  • src 속성은 사용하지 말 것

사용(작성) 위치에 대한 주의 사항

  • 일반적으로 head 태그 안에 작성
  • html 태그 요소를 제어하는 경우 head 안의 코드는 동작하지 않을 수 있음. => 브라우저는 head의 내용으 모두 로드할 때 까지 body 요소를 렌더링 하지 않기 때문
    • html 요소가 렌더링이 완료된 body 종료 태그 윗 줄에 작성
    • 요소가 모두 로드된 후 스크립트를 실행하라는 함수 사용 => JS
    • *** src로 연결된 파일은 defer 속성을 이용

자바스크립트 기초 문법

  • 객체.메소드() => 자바스크립트 함수, 행동
  • 객체.속성(=프롬퍼티) => 객체의 속성

자바스크립트 출력

  • 경고창: window.alert('hello')
  • 개발자 도구 콘솔 : conslole.logo(3+ 8)
  • html 요소에 출력 : document.getElemntByld('id-name').innerHTML = 'hello'
